# Context-Driven AI Development (CDAD)

> **When context doesn't govern AI, AI governs the solution.**

An AI-assisted software development methodology based on governed context, architectural control, knowledge traceability, and sustainable acceleration.

## What is CDAD?

CDAD is a methodology that treats governed context as the primary source of truth for AI-assisted software development.

Its objective is to accelerate implementation without sacrificing architectural integrity, technical consistency, or knowledge traceability.

---

## Core Pillars

- Context Engineering
- Governance of Context
- Context Protection Pattern (CPP)
- Markdown-Driven Development
- Agentic Development Environment (ADE)
- AI-Assisted Delivery
- Controlled Vibe Coding
